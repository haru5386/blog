## Getting Started

To get started with the blog locally, follow these steps:

1. **Install dependencies**:
   ```bash
   bun install
   ```

2. **Run the development server**:
   ```bash
   bun run dev
   ```

3. **Open your browser** and visit `http://localhost:3000` to see the blog in action.
